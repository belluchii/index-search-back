# Meli Index Search

Search index of 850,000+ Amazon products with full-text search capabilities.

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Fastify
- **Database:** MongoDB (Mongoose)
- **Environment:** dotenv

## Requirements

- Node.js >= 18.x
- MongoDB

## Installation
1. Clone the repository:
```bash
git clone https://github.com/valentin/meli-index-search.git
cd meli-index-search
```
2. Install dependencies:

```bash
npm install
```
3. Create a .env file:
```env
PORT=3000
MONGODB_URI=mongodb://localhost:27017/meli-index-search
```
4. Start the development server:
```bash
npm run dev
```
